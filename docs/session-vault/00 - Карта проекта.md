# 🎞️ «Кадр» — карта проекта

Событийная камера для РФ (аналог pov.camera). Гость заходит по QR/ссылке **без установки**, снимает на «плёночную» камеру, кадры летят в общую галерею события. Хост — на iOS. Полное соответствие **152-ФЗ**.

> [!info] Это Obsidian-vault с конспектом рабочей сессии
> Сохраняет ход разработки, решения и грабли. Источник истины по продукту — `SPECIFICATION.md` и `PROJECT_IDEA.md` в корне репозитория.
