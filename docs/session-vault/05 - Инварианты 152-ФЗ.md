# 05 — Инварианты 152-ФЗ

[[00 - Карта проекта|← к карте]]

> [!danger] Жёсткие правила — нарушать НЕЛЬЗЯ
> Проверяет security-agent. Любой код, нарушающий их, считается багом и блокируется.

1. **Локализация (ст. 18.5):** Postgres + Storage + деплой — только в РФ (self-hosted Supabase в Yandex/VK Cloud). Запрещены: Supabase Cloud, Vercel, Firebase, Stripe, Google Analytics, зарубежные CDN/шрифты.
2. **Никакого распознавания/детекции лиц** — нигде. Фильтры в `lib/filters.ts` — только пиксельная арифметика.
3. **RLS включён на КАЖДОЙ таблице.** Нет таблицы без политик.
4. **Аналитика только российская** (AppMetrica/Метрика). Нет Google/Firebase SDK.
5. Фото отдаются **ТОЛЬКО подписанным URL с TTL** (≤600с). Публичных бакетов нет. `toPublicUrl` меняет только хост, подпись/TTL сохраняются.
6. **Согласие** пишется в `consents` (subject_uid, type, purpose, policy_version, granted_at, IP, UA) **ДО первого кадра гостя**. Чекбокс не предзаполнен; кнопка заблокирована без галочки.

## Как проверено в сессии

- security-agent по фазе 2 и фазе 3 → **вердикт PASS** по инвариантам (нет critical/high нарушений).
- consent реально пишется до создания гостя (`join-event`), с IP/UA из заголовков — подтверждено в БД.
- фильтр применяется на клиенте, оригинал не грузится — подтверждено в `camera/page.tsx`.
- бакет приватный, отдача только через `photo-url` под service-role.

## Auth локально (без нарушения инвариантов)

- Анонимные сессии гостей включены (`enable_anonymous_sign_ins=true`).
- Хост локально — **email-OTP** через Inbucket (наружу ничего не уходит). На проде — российский SMS (SMS Aero/SMSC) через Auth hook.
- Twilio-блок в `config.toml` — неактивный дефолт CLI (`enabled=false`), помечен комментарием.

## Открытые пункты (для прод-аудита)

- DELETE-политика на `storage.objects` для отзыва согласия + retention (`purge-expired`) — фаза 4/5.
- Доверенный источник IP под РФ-прокси (сейчас `x-forwarded-for`).
- Сузить CORS `Allow-Origin: '*'` до домена PWA на проде.
- Серверная валидация content-type/размера файла. См. [[07 - Найденные баги (бэклог)]].
